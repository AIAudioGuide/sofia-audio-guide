# Sofia Audio Guide 🎧🗺️

An interactive AI-powered audio guide for exploring Sofia, Bulgaria.

## Live App
**https://sofia-audio-guide.vercel.app**

## Features
- KrisVoice audio narration (ElevenLabs)
- Interactive map (Mapbox)
- AI chat assistant (OpenAI)
- Local places guide
- Full Bulgarian history intro

## Tech Stack
- Next.js + Tailwind CSS
- Mapbox
- ElevenLabs
- OpenAI
- Vercel

## Quick Start
```bash
npm install
npm run dev
```
